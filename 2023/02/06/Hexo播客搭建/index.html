# Hexo 博客搭建教程

# 第一步：安装软件

  1. 安装 git (省略)
  2. 安装 node (省略)

# 第二步：软件安装完准备环境

  1. node 环境变量

    1. 在 node 安装目录下新建 node_global,node_cache 文件夹

    1. 新建系统环境变量

    1. 用户变量 path 添加

  2. 更改缓存文件和模块默认存放位置

  3. node 修改为淘宝镜像源

# 第三步: cnpm 安装 hexo

​ 安装完后可以用 hexo -v 查看安装情况

# 第四步：创建博客目录

  1. 新建一个 blog 目录，并进入到该目录下

  2. 初始化一个博客

  3. 启动博客预览

  4. 复制启动后的地址到浏览器打开

  5. 新建一篇文章


    进入到文章所在的位置，\source_posts, 编辑文章内容

  6. 清理缓存并重新生成

    hexo c
//c = clean

    hexo g
//g = generate

  7. 启动重新生成的网站

    hexo s
//s = server

# 第五步：部署博客

  1. 我们不能一直在本地看对吧，所以我们可以免费部署到 github 上

  2. 进入 github 官网，新建仓库，仓库名字必须符合规则， 你的 github 昵称 + github.io, 然后点击 create 创建

  3. 上传文件到 github 仓库

    用管理员权限打开 git bash 窗口

    执行下列操作

  4. 上传完毕后访问网站

  5. 当然了，这个只是最基本的搭建一个博客网站，主题是默认的，当然也是可以选择更加花里胡哨的主题，下篇文章再说
