# 第二章 HTML 语法基础

# 1. meta 标签

meta 标签是 HTML 语言中的一种标签，用于描述 HTML 文档的元数据（metadata）。元数据是关于数据的信息，比如网页的标题、作者、关键词、编码方式、刷新时间、视口等信息。

常用的 meta 标签包括：

  • meta charset ：用于指定 HTML 文档的字符集编码方式。

  • meta name="viewport" ：用于指定移动设备的视口（viewport），以便适配不同分辨率的设备。

  • meta name="keywords" ：用于指定网页的关键词。

  • meta name="description" ：用于指定网页的描述信息。

  • meta http-equiv="refresh" ：用于设置页面的刷新方式和时间间隔。

使用 meta 标签可以优化网页的搜索引擎排名、提升网页的用户体验、加快网页的加载速度等。

# 2. 常用的文本标签

  1. <p> 标签：用于表示段落，通常与文本内容一起使用。

  2. <h1> ~ <h6> 标签：用于表示标题，分别表示从大到小的六个级别，其中 <h1> 最大， <h6> 最小。

  3. <a> 标签：用于创建超链接，可以将文本或图片链接到其他页面或文件。

  4. <strong> 或 <b> 和 <em> 或 <i> 标签：用于表示强调和重点内容，分别表示加粗和斜体效果。

  5. <br> 标签：用于表示换行，可以在不创建新段落的情况下将文本分行显示。

  6. <ul> 和 <li> 标签：用于表示无序列表， <ul> 表示列表本身， <li> 表示每个列表项。

  7. <ol> 和 <li> 标签：用于表示有序列表， <ol> 表示列表本身， <li> 表示每个列表项。

  8. <blockquote> 标签：用于表示长引用文本，可以缩进显示。

  9. <pre> 标签：用于表示预格式化文本，保留所有空格、换行和标签。

  10. <code> 标签：用于表示代码块，通常与 <pre> 标签一起使用，以保留代码格式和缩进。

  11. <sup> 和 <sub> 标签：用于表示上标和下标，分别表示上方和下方的小字符。

  12. <del> 和 <ins> 标签：用于表示删除和插入的文本，通常用于比较文本变化的情况。

# 3. 常用的多媒体标签

  1. <img> ：用于显示图片。

  2. <audio> ：用于播放音频文件。

  3. <video> ：用于播放视频文件。

  4. <source> ：为 <video> 和 <audio> 元素定义多个媒体资源，以便在不同的浏览器和设备上进行兼容。

  5. <iframe> ：用于在当前网页中嵌入另一个网页。

  6. <a> 标签，用于创建链接到其他网页、文件、位置等的超链接。以下是 <a> 标签的常用属性及用法：

    • href 属性：指定链接的目标地址，可以是一个 URL、一个文件路径、一个锚点等。

    • target 属性：指定链接打开的方式，可以是在当前窗口打开、在新窗口打开等。

    • rel 属性：指定链接与当前页面的关系，用于搜索引擎优化（SEO）。
