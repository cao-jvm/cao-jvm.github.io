# 选择题

  1. Catalyst6500 配置 IP 地址的命令格式: set interface sc0<ip_addr><ip mask><ip addr>

  2. IEEE 制定的生成树协议标准是 IEEE802.1D

  3. VLAN 工作在 OSI 参考模型的网络层

  4. Loopback (环回接口) 作为一台路由器的管理地址，网络管理员为其分配一个 IP 地址作为管理地址，掩码应为 255.255.255.255

  5. D 类系统不能用于度用户环境下的重要信息处理

  6. ICMP 消息封装在 IP 数据包内

  7. 网络防火墙不能够阻断的攻击是 SQL 注入

  8. 用于检测本机配置的域名服务器是否工作正常的命令是 pathping, 可以用于检测网络联通情况可和探测到达目的计算机的路径

  9. A 类地址段是：1.0.0.0-127.255.255.255;B 类地址段是 128.0.0.0-191.255.255.255;C 类地址段是 192.0.0.0-223.255.255.255

  10. 网络地址是 IP 地址中网络位不变，主机位置为 0 的地址；直播广播地址是 IP 地址网络位不变，主机位置为 1 的地址；主机号是 IP 的地址网络位置为 0, 主机位不变的地址；子网内的最后一个可用的 IP 地址的直接广播地址的前一个地址

    例子: IP 地址为 111.181.21.9 , 子网掩码为 255.192.0.0

    网络位和主机位是由子网掩码来确定，在子网掩码中，网络位全为 1, 主机位全为 0

    把 IP 地址和子网掩码全部转换成二进制，然后按位进行与运算，即可得到网络地址

    111.181.21.9: 01101111.10110101.00010101.00001001

    255.192.0.0: 11111111.11000000.00000000.00000000

    从子网掩码得出，前十位是网络位，后 22 位是主机位，经过按位与运行后得：01101111.100000000.00000000.00000000, 转换成十进制为 111.128.0.0, 所以网络地址为 111.128.0.0

    直接广播地址是 IP 地址网络位不变，主机位置为 1 的地址，将 IP 地址中的后 22 位全部变为 1. 得到 01101111.10111111.11111111.11111111, 转换为十进制为 111.191.255.255, 所以直接广播地址为 111.191.255.255

    主机号是 IP 地址网络位全为 0, 主机位不变的地址 00000000.00110101.00010101.0001001, 转换为十进制为 0.53.21.9, 所以主机号为 0.53.21.9

    子网内的最后一个可用的 IP 地址是直接广播地址的前一个地址，上面的直接广播地址为 111.191.255.255, 所以最后一个可用的 IP 地址是 111.191.255.254

  11. 不属于无线接入技术的是: APON

  12. HFC 通过 Cable Modem 把用户计算机与有线电视同轴电缆连接起来

  13. 典型的高端路由器的可靠性与可用性指标应该达到:

    • 无故障连续工作时间 (MTBF) 大于 10 万个小时
    • 系统故障恢复时间小于 30 分钟
    • 系统具有自动保护切换功能，主别用切换时间小于 50 毫秒
    • SDH 与 ATM 接口自动保护切换功能，切换时间小于 50 毫秒
    • 主处理器，主存储器，交换矩阵，电源，总线管理器与网络管理接口等主要部件需要有热插拔冗余备份，并提供远程测试诊断能力
    • 路由器系统内部不存在单点故障

  14. 内部网关以协议主要有 RIP 和 OSPF 协议.BGP 是边界网关协议

  15. 每一个 OSPF 区域拥有一个 32 位的区域标识符，每一个区域内的路由器数不超过 200 个

  16. FTP 服务器的选项中，不提供 IP 访问选项的是服务器选项

  17. 下列安全设备中，不能部署在网络出口的是网络版防病毒

  18. RPR 用统计复用的方法传输 IP 分组

  19. 802.16a 用于固定节点接入

  20. ADSL 上行传输速率为 64kbps-640kbps

  21. 采用 RAID 技术一定程度上可以提高磁盘存储容量但是不能提高容错能力

  22. 不同 AS 之间使用的路由协议是 BGP

  23. 中继器工作在物理层

  24. 交换机根据接受数据包中的 MAC 地址过滤和转发数据

  25. 不属于宽带城域网 Qos 保证技术的是 PSTN

  26. 若服务器系统可用性达到 99.99%，那么系统平均无故障时间（单位：分钟）约为：52547 MTBF<52547.44

  27. 基于网络的入侵防护系统可以阻断的攻击手段是 Smurt 攻击，Smurt 是冒充受害主机的 ip 地址，向一个大的网络发送定向广播包

  28. OSPF 协议中，每一个区域 OSPF 拥有一个 128 为位的区域标示符

  29. Catalyst 3500 交换机配置优先级的命令是在全局模式下输入 spannig-tree vlan priority <0-61440>

  30. IEEE802.1Q 标准规定，VLAN ID 用 12 比特表示

  31. 蓝牙技术指标跳频速率为 1640 次 / 秒

  32. 不属于网站选项是带宽限制

  33. VDSL 上行速率为 2.3Mbps，下行速率为 51Mbps

  34. 在蓝牙系统当中，密钥以 8bit 为单位增长，最长 128bit

  35. 客户机添加排除连续地址时必须输入起始 ip 地址和结束 ip 地址

  36. 蠕虫病毒不需要宿主程序，他是一段独立的程序或代码

  37. 可运营以太网的设备和线路必须符合电信网络 99.999% 的高运行可靠性

  38. 丢包率是衡量路由器超负荷工作时的性能指标之一

  39. OSPF 和 RIP 协议都采用了 “最短路径有限” 的原则

  40. 在网络中部署 IDS 时，可以使用集线器替代分路器（TAP）

  41. 蓝牙技术的同步信道速率是 64kbps（3 个全双工通道）

  42. IEEE802.11a 的最大容量是 432Mbps

  43. netflow-e 命令用于设置发送频率；net statistics 命令用于显示本地工作站或服务器的统计日志；nbstat-s 命令用于列出会话及其目的 ip 地址；netstat -e 命令用于显示以太网的统计信息

  44. Bridge ID 值最小的成为根网桥和根交换机

  45. 入侵检测系统结构中，出现单点故障影响最严重的是集中式

  46. OPNET 不能用于网络嗅探

  47. 添加新建域时输入的域名可以是合格的域名或者是其他任何描述

套路技巧

地址类别 A 类：1-127
B 类：128-191
C 类：192-223
网络地址 网络位不变，主机位变 0
直接广播地址 网络为不变，主机位变 1

子网内第一个可用 IP 地址 网络地址 + 1
子网内最后一个可用 IP 地址 直接广播地址 - 1
受限广播地址 255.255.255.255

第一步：子网掩码转换成二进制，划分 1，0 分界（网络位与主机位）

11111111.111 | 00000.00000000.00000000 -----> 所以 ip 地址的前 11 位为网络为，后面的为主机位

第二步：ip 地址转换成二进制：分开网络位和主机位

011111101.101 |01111.00010100.00000111 ----> 按照规律，前 11 位是网络位，后面是主机位

# 综合题第二题

  1. ip route 模板

    阴影边缘路由器（R1,R2,R3)

    Ip route 0.0.0.0 0.0.0.0 <下一跳路由器 Ip 地址>

    阴影外路由器 (R4)

    Ip route <阴影 ip 地址（如果有多个地址要先聚合）> < 阴影区子网掩码 > < 下一跳路由器 IP 地址 >

    例如：

    对于 R3

    Route- R3(config)#ip route 0.0.0.0 0.0.0.0 211.68.69.169

    对于 R4

    Route-R4(config)#ip route 211.89.23.0 255.255.255.0 211.68.69.170

  2. bandwidth 模板

    bandwidth <带宽> 单位： kbps

    1Gbps = 1000Mbps = 1000000kbps

    例如： 某园区用 10Gbps 的 POS 技术与 Internet 相连

    Route-R3(config-if)#bandwidth 10000000

  3. crc 模板（crc 校验位）

    crc<32/16> (题干未说明情况下为 32)

    例如： Router- R3 (config-if)# crc 32

  4. ip address 模板

    ip address <这个路由器的 ip 地址><这个路由器的子网掩码>

    例如：

    对于 R3

    Router-R3(config-if)#ip address 211.68.69.170 255.255.255.252

  5. pos framing 模板（帧格式）

    pos framing <sdh/sonet>（根据题干要求选择填写）

    例如： POS 接口的帧格式是 SDH

    Router-R3(config-if)# pos framing sdh

  6. pos flag 模板

    pos flag <s1s0 2>/<s1s0 0> (<s1s0 2>代表 sdh，<s1s0 0>代表 sonet)

    例如： POS 接口的帧格式是 SDH

    Router-R3(config-if)# pos flag s1s0 2

  7. lease 模板（租用时间）

    lease <天>[时] [分] （天不可以省略）

    例如： 地址租用时间为 5 小时 30 分

    Router-R1(dhcp-config)# lease 0 5 30

  8. network 模板

    network <阴影区 ip 地址><阴影区子网掩码反码>

  9. area 模板

    aera 0 range <阴影区 ip 地址> <阴影区子网掩码>

    8，9 这两个模板同时出现了，相互对应

    例如：

    Router-R3(config-router)# network 221.89.23.0 0.0.0.255 area 0

    Router-R3(config-router)#area 0 range 221.89.23.0 255.255.255.0

  10. router ospf 模板

    router ospf <进程号> （出现在 network 命令前；进程号一般已知）

    例如：

    Router-R3(config)# router ospf 63

    Router-R3(config-router)# network 221.89.23.0 <0.0.0.255> area 0

  11. excluded-address 模板（排除不用于动态分配的地址）

    ip dhcp excluded-address [high address]

    （low address 不可省略，high address 可省略）

    例如： 221。89.23.200-221.89.23.254 不用于分配

    Router-R1(dhcp-config)# ip dchp excluded-address 221.89.23.200 221.89.23.254

  12. 建立 IP 地址池

    ip dhcp pool <地址池名>

    例如：

    Router-R1(config)#ip dhcp pool addresspool

  13. network 模板（DHCP 下）

    network <地址池地址> <地址池正子网掩码>

    与 ospf 下的 network 命令区分

    例如：

    Router-R1(dchp-config)#network 121.193.130.0 255.255.255.0

  14. 配置默认网关（缺省网关）

    default-router <缺省网关地址>

    例如：Router-R1 (dchp-config)#default-router 121.193.130.1

  15. 配置域名服务器

    dns-server address <主地址> <辅地址>

    例如：

    Router-R1(dhcp-config)# dns-server address 121.193.129.27 121.193.129.26

  16. 打开端口

    no shutdown

    例如：

    Router-R3(config-if) #no shutdown(打开端口)

  17. 配置异步串行接口

    async default ip address <接口地址>

    例如：

    Router(config-if) async default ip address 202.113.7.16

  18. ip address 模板（配置 loopback 接口情况下）

    ip address <接口的 ip 地址> <255.255.255.255>

    例如：

    Router-R3(config-if)# ip address 192.167.166.6 255.255.255.255

    ​ 上面 4.ip address 模板（没有配置 loopback 接口情况下）

    ​ ipaddress <这个路由器的 ip 地址> <这个路由器的子网掩码>
